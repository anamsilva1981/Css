Seletores

Seletores de Tipo: Este seletor todas as tags de um mesmo tipo. quando [e definiodo o estilo a TAG < p > estará definido para todas as TAG < p >


Seletor Descendente: Pode selecionar um ou mais elementos que estão dentro de outro, ou seja, que são descendentes do elemento principal. Exemplo: "p span". Com isso, selecionamos apenas tags "span" que estão dentro de parágrafos

Pseudo-Classes: São tipo se classes especiais que não são definidas pelo desenvolvedor (já são pré-definidas). Por exemplo, "a:hover"


id(#): Uma identificação única: só pode ser utilizada uma vez no documento inteiro. Normalmente é utilizada para identificar elementos estruturais da página "div id="cabecalho"

Classes(.): Pode se repetir na página e tambem combinar-se com outras (pode ser colocada mais de uma classe em um elemento) "p class="titulo"
